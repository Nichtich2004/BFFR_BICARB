# BFFR Bicarb Pouches — Website

## GitHub Pages Setup

1. Push this entire folder to a GitHub repository
2. Go to **Settings → Pages**
3. Under "Source", select **Deploy from a branch**
4. Choose branch: `main`, folder: `/ (root)`
5. Click **Save**

Your site will be live at: `https://yourusername.github.io/repo-name/`

## Files
- `index.html` — Main landing page (vision, science, team)
- `shop.html` — Shop with Discord webhook order system
- `assets/images/` — All logos and photos
- `.nojekyll` — Disables Jekyll so GitHub Pages serves files as-is
